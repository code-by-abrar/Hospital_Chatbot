CareBot is a lightning-fast, AI-powered virtual assistant developed using real-world healthcare data from the **Pakistan Institute of Medical Sciences (PIMS)**. It helps patients get instant, accurate answers regarding PIMS hospital policies, doctor schedules, OPD timings, and appointment details with near-zero latency.

## ‚ú® Key Features
* **Real-World Data Grounding:** Specifically engineered to retrieve and process complex medical and administrative data from **PIMS Hospital**, preventing AI hallucinations.
* **Lightning Fast Inference:** Powered by the **Groq API**, delivering ultra-fast response times for a seamless user experience.
* **LangChain Powered:** Utilizes **LangChain** for robust prompt management, context structuring, and LLM chain execution.
* **Custom UI:** A sleek, responsive, and user-friendly frontend built with HTML/CSS/JS that renders Markdown responses beautifully.
* **Asynchronous Backend:** Built with **FastAPI** to handle multiple patient requests efficiently and smoothly.
